# Srishti

Srishti (Creation) is a premium real-time collaborative whiteboard application designed for the modern vizualizer. Built with a "Cosmic Dark" aesthetic, it offers an infinite canvas for your wildest ideas.

## üöÄ Key Features

*   **Infinite Canvas**: A vast workspace to unleash your universe of ideas.
*   **Real-time Collaboration**: Multiple users can draw, move, and interact simultaneously.
*   **Authentication & Organizations**: Secure login and multi-tenancy support using **Clerk**.
*   **Premium Tools**: Pen, shapes, sticky notes, and text with glassmorphic UI.
*   **Dark Mode First**: A deep, immersive interface designed for focus.
